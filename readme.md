## Getting Started

```
.
├── package.json
├── package-lock.json
├── readme.md
└── summer-sale
    ├── assets
    │   ├── css
    │   │   └── main.css
    │   ├── images
    │   └── js
    │       └── main.js
    ├── figma
    │   
    └── index.html

```

1. Clone this repository

```shell
git clone https://github.com/mrsnailo/js-practice-projects.git
```

2.  Open terminal  , and cd into any project folder , then   install dependencies. 

```bash
cd js-practice-projects &&
npm install
```

3. Then run the poject 

```shell
npm start
```
